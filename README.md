> Welcome to Week 3! This is a big one.
>
> In Week 2, we learned about comparison operators (`==`, `>`, `<` ...) and logical operators (`and`, `or`, `not`). But we only **printed** their results — `True` or `False`.
>
> Today, those `True`/`False` results finally start to **do something**. This is called **control flow** — letting our program *decide* what to do.
>
> **What you'll learn today:**
>
> - The `if` statement
> - `if` / `else` — two paths
> - `if` / `elif` / `else` — many paths
> - Indentation (Python's block rule)
> - Logical operators inside conditions (`and`, `or`, `not`)
> - Membership in conditions (`in`)
> - Truthy / falsy values
